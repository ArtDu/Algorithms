##### Без повторении
    
    Берем и уменьшаем вес или не берем
    dp(i, w)
    
    dp(0, w) = 0
    dp(i, w) = 0
    dp(i, w) = max(dp(i - 1, w), dp(i - 1, w - wi) + pi)
    
    
    Рекурентная формула:
        dp(i, w) = 0                                         , i = 0 or w = 0 
        dp(i, w) = max(dp(i - 1, w), dp(i - 1, w - wi) + pi) , i != 0 and w != 0
